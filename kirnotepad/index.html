KirNotepad

Текстовый редактор на Python с поддержкой плагинов

Лёгкий, быстрый и расширяемый текстовый редактор, написанный на Python с использованием Tkinter. Поддерживает систему плагинов — добавляй функционал, как тебе нужно.

Особенности

  • Поддержка плагинов (.py)
  • Синтаксис подсветки (через плагины)
  • Тёмная тема по умолчанию
  • Кроссплатформенность
